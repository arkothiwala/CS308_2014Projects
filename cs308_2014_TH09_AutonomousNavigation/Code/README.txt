SETUP
------------------------

A. Keil uVision Project
-- Keil uVision4 is required.
-- Just run the uVision4 project file. Compile to get .bin file which is to be copied onto the bot.


B. Android app
-- Android Studio is required. (http://developer.android.com/sdk/installing/studio.html)
-- Open the AutoNav project in Android Studio. Build to get app.apk, which is to installed on the mobile device used with the bot.

Title of Project : Tomato Plucker
---------------------------------------------------------------
Brief Description :

Aim of our project is to automate the process of mapping and 
plucking ripe tomatoes in a particular green house setting
---------------------------------------------------------------
Softwares and Prerequisites:

1.AVR Studio 
2.AVR Bootloader 
3.Python
4.OpenCV
5.pygame

1 and 2 can be installed just by running their .exe files and 
proceeding on prompts(folder given in lab)

3 and 4 can be installed by following links
http://docs.opencv.org/trunk/doc/py_tutorials/py_tutorials.html

5 can be installed from
http://www.pygame.org/download.shtml

----------------------------------------------------------------
Instructions to Setup:

1.Open Project in AVR studio, Build (F7)
2.Using AVR Bootloader, burn the tomato_plucker_erts.hex file on 
  to the bot
3.Run the python script on the server
4.Connect zigbee module to server
5.Connect camera to server(USB camera)
6.Run Python script(server.py) on the server
7.Keep the bot at 0,0 position on the grid
8.Switch on the bot
